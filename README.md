## Overview
Interactive Power BI dashboard analyzing coffee shop sales performance across time, products, and store locations.
All data modeling and calculations were performed inside Power BI using DAX.

## Dataset
Transactional sales data in CSV/Excel format including date, time, product details, store location, quantity, and price.

## Tools Used
- Power BI Desktop
- DAX
- Power Query
- Excel / CSV

## Dashboard Highlights
- KPI cards for Total Sales, Total Orders, and Total Quantity Sold with MoM comparison
- Sales trend analysis by day and hour
- Product category and product type performance
- Store-wise sales analysis
- Weekday vs weekend sales distribution
- Interactive slicers and custom tooltips

## Purpose
This project demonstrates practical Power BI, DAX, and data modeling skills for data analyst and business intelligence roles.
